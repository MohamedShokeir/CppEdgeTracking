# CppEdgeTracking

EdgeTracking (ET) is an image analysis code that detects and calculates the minimum distance of between 2 edges.

![GitHub Logo](./logo.png = 250x250)

## Installation
```bash
git clone https://github.com/MohamedShokeir/CppEdgeTracking.git
cd CppEdgeTracking/
make build
```

## Usage
```bash
./build ET file.txt
```
